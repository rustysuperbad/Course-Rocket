# Course-Rocket
CourseRocket.ai is a web application that makes custom courses completely designed based on what the user is working on, or wants to learn.
Main functionalities of the application -

1> accquiring youtube videos in the order of relevance of learning. - youtube api
2> research papers sourced from throughout the internet. - papers api
3> ai- generated course content that helps in learning. - groq api

authentication powered by firebase.
